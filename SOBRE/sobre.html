Alfabetização e letramentos digitais - desenvolvimento de software para plataforma web para alfabetização na EJA
Objetivo:

Desenvolver aplicativo para plataforma web voltado para auxiliar na alfabetização em turmas de EJA (através de atividades pedagógicas de alfabetização de adultos, com exercícios), avaliando resultados em comparação com turmas que não usarão o recurso digital.


A aplicação é dividida em cinco módulos, onde são apresentados diversos temas, de forma progressiva, iniciando com a apresentação do alfabeto, formação de sílabas a partir das letras, compreensão de frases, práticas de escrita de palavras entre outras coisas.
